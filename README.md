## 🛠️ How to Run

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Open in IntelliJ IDEA** (or any Java-compatible IDE like Eclipse or VS Code with Java extensions).

3. Navigate to the `src/` folder and open any `.java` file.

4. Each file typically has a `main()` method — simply run it directly to test the solution.

> **Note:** No external dependencies required. Pure Java (JDK 8+).
